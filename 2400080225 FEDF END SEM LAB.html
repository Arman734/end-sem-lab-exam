Git Assignment: Feature Branching & Conflict Resolution

Task:

  1. Create two branches:
    • feature-course – add Course Registration feature
    • feature-event – add Event Calendar feature
  2. Implement each feature in its respective branch.
  3. Merge both branches into main.
  4. If a merge conflict occurs, resolve it and make a final commit.

Example code added in each feature:

feature-course:

<div id="course-registration">
  <h3>Course Registration</h3>
  <input type="text" placeholder="Enter Name">
  <button>Register</button>
</div>

  </ul>
